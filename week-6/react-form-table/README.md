# React Form Table
## About

- Add users with username, email, and date of birth
- Form validation for username input
- Display submitted users in a responsive table
- Built with React, Vite, Tailwind CSS, and React Hook Form


## Project Structure

- `src/App.jsx` - Main application entry
- `src/components/From.jsx` - User form and table component
- `src/main.jsx` - Vite entry point
- `src/index.css` - Project styles
- `App.css` - Component-specific styles

## Usage

1. Enter a username, email, and date of birth.
2. Click **Add User**.
3. The user will be added to the table below the form.
