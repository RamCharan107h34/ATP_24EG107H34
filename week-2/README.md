# Week 2

This folder contains JavaScript practice tasks for Week 2. The files are organized by topic and represent a series of hands-on exercises using array methods, object handling, and small project simulations.

## Contents

- `1-01-daily-temperature-analyzer.js` - Daily temperature analyzer using `filter`, `map`, `reduce`, `find`, and `findIndex`
- `1-02-online-course-name-processor.js` - Online course name processor for formatting course lists
- `1-03-student-marks-list.js` - Student marks list processing with array filtering and transformations
- `2-01-shopping-cart-summary.js` - Shopping cart summary for an e-commerce website
- `2-02-student-performance-dashboard.js` - Student performance dashboard and result analysis
- `2-03-employee-payroll-processor.js` - Employee payroll processor for salary calculations
- `2-04-movie-recommendation-system.js` - Movie recommendation system simulation
- `2-05-bank-transaction-analyzer.js` - Bank transaction analyzer for credit/debit processing
- `reduce-min-max.js` - Reduce-based min/max finder
- `library-book-management.js` - Library book management system using a `Book` class
- `array-copy-extend.js` - Array copy and extend example with spread syntax
- `user-object-update.js` - User object update and cloning example
- `rest-sum-function.js` - REST parameter sum function example
- `shallow-copy-use-case.js` - Shallow copy use case demonstration
- `deep-copy-use-case.js` - Deep copy use case demonstration
- `exam-submission-simulator.js` - Exam submission simulator example
- `otp-countdown-simulator.js` - OTP countdown simulator example

## Notes
