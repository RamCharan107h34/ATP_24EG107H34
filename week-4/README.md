# Week 4 HTML Tasks

This folder contains three HTML practice files for a week 4 assignment.

## Files

- `task01.html`
  - A simple HTML page with paragraphs, a heading, a link, and an unordered list.
  - Demonstrates basic HTML structure: `<p>`, `<h1>`, `<a>`, and `<ul>`/`<li>` elements.

- `task02.html`
  - An example of nested lists in HTML.
  - Includes unordered list with nested lists for fruits and vegetables.

- `task03.html`
  - A small profile page for "Pochi the cat." 
  - Uses headings, paragraphs, an image, an ordered list, and links.

## How to view

1. Open one of the files in a web browser.
2. Alternatively, open the file in a code editor and use live preview if available.
