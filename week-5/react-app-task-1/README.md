# React App Task 1

A simple React application built with Vite and Tailwind CSS that renders a product grid using static product data.

## Project Overview

- Uses React 19 and Vite 4+ for development and production builds.
- Includes Tailwind CSS for styling and responsive layout.
- Renders a list of product cards from static data defined in `src/App.jsx`.
- Displays product title and description using a reusable `Product` component.

## Scripts

- `npm run dev` - start the Vite development server

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal to view the app.
