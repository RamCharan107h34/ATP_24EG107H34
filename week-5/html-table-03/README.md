# Table Task 03

This project is a simple front-end exercise that demonstrates how to create and style a table layout using HTML and CSS.

## About

- `index.html` includes the page structure and the table content.
- `style.css` provides styling for the table, including layout, borders, spacing, and colors.

## Files

- `index.html` - the HTML document with the table markup.
- `style.css` - the CSS stylesheet that controls presentation and layout.

## Usage

1. Open `index.html` in a web browser.
2. See the rendered table and page styling.
3. Edit `index.html` to change table content or structure.
4. Edit `style.css` to modify colors, spacing, borders, or layout.

## Notes

- This is a web page example only, with no JavaScript or server-side code.
- Refresh the browser after saving changes to see updates.
