# Table Task 02

This project is a simple web page with an HTML table layout and custom styling.

## Files

- `index.html` — main HTML file for the page structure.
- `style.css` — CSS file for styling the table and page.

## Usage

1. Open `index.html` in a web browser.
2. The page will render the table with styles from `style.css`.

## Notes

- Modify `index.html` to change table content.
- Update `style.css` to adjust colors, spacing, and layout.
