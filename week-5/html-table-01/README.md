# Table Task 01

This project is a simple web page with an HTML table layout and styling.

## Files

- `index.html` - The main HTML page.
- `style.css` - Styles for the page and table.

## Usage

1. Open `index.html` in a web browser.
2. View the table layout and styling.

## Notes

- This repository contains a small static frontend example.
- You can edit `index.html` and `style.css` to customize the table content and appearance.
